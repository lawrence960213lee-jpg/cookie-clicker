# 🌠 餅乾宇宙帝國 (Cookie Clicker - Cloud Edition)

這是一個基於 HTML5、CSS3 與 JavaScript 開發的點擊冒險遊戲，並結合了 **Google Firebase** 雲端技術，支援全台灣（甚至金門全境！）玩家即時同步產量與全球排行榜競爭。

## 🚀 遊戲特色

* **雲端同步系統**：使用 Firebase Realtime Database，換電腦玩也不怕紀錄消失。
* **雙重遊戲模式**：
    * **無盡模式**：累積產量，建立屬於你的宇宙帝國。
    * **競速模式**：挑戰最快達成 1,000 餅乾，爭奪計時排行榜第一名。
* **現代化 UI**：採用 **Glassmorphism (玻璃擬態)** 風格，搭配宇宙星空動態背景。
* **流暢體驗**：優化點擊邏輯，防止文字反白干擾，並加入即時點擊浮動特效。

## 🎮 如何遊玩

1.  **進入遊戲**：打開 [你的網頁連結](https://lawrence960213lee-jpg.github.io/cookie-clicker/)。
2.  **建立身分**：輸入指揮官代號與密鑰，點擊「建立身分」。
3.  **開始點擊**：點擊中間的大餅乾開始生產。
4.  **升級帝國**：使用賺到的餅乾購買「強化滑鼠」或「聘僱奶奶」自動生產。
5.  **查看排行**：點擊「全球排行榜」查看你目前的帝國排名。
